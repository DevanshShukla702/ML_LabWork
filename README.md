# Machine Learning Lab Work 🔬

Welcome to the **Machine Learning Lab Work** repository! This repository contains a structured, comprehensive suite of practical machine learning experiments implemented in Python. It spans from foundational data processing and introductory classification mechanisms to advanced ensemble learning, unsupervised clustering, and introductory deep learning/NLP concepts.

## 🚀 Introduction
The core purpose of this repository is to track a series of hands-on Machine Learning lab assignments. Each lab is contained within its own directory and corresponds to a Jupyter Notebook that solves specific data problems using popular data science libraries like `scikit-learn`, `pandas`, `matplotlib`, `tensorflow`/`keras`, and `nltk`. 

---

## 📂 Experiments Directory

### **Lab-01: Exploratory Data Analysis & Setup** 📊
- Basic data manipulation and setup. This lab explores initial exploratory data analysis concepts, learning how to handle basic structures like DataFrames.

### **Lab-02: Data Preprocessing & Visualization** 🧹
- Introduces essential data preprocessing techniques along with foundational data visualization utilizing scatter plots and correlation matrices.

### **Lab-03: Model Building Basics** 🏗️
- Demonstrates foundational model setups, pipeline processing, and standardized Train/Test set splits.

### **Lab-04: Linear & Logistic Regression** 📈
- Implements core statistical machine learning models. Covers predictive Linear Regression models for continuous outputs and Logistic Regression models for binary classification boundaries.

### **Lab-05: Decision Trees & Regressors** 🌲
- Focuses on non-linear predictive mechanisms. Explores Decision Tree classifiers alongside encoding mechanisms for categorical targets, followed by Decision Tree Regressors.

### **Lab-06: Naïve Bayes & Multi-Model Comparison** ⚖️
- Compares probability-based Naïve Bayes Classifiers side-by-side with Decision Trees, rigorously comparing accuracy metrics and classification reports.

### **Lab-07: Ensemble Learning** 🤝
- Advanced classification leveraging multiple estimators:
  - **Bagging** (Bootstrap Aggregating)
  - **Boosting** (AdaBoost & Gradient Boosting)
  - **Stacking** (Composing a Meta-Learner over KNN, Logistic, and DTs)

### **Lab-08: Unsupervised Learning & Clustering** 🧩
- Dives into grouping uncategorized data.
  - **K-Means Clustering**: Partitioning data into spatial clusters.
  - **Hierarchical Clustering**: Agglomerative clustering visualized using Dendrograms.

### **Lab-09: Artificial Neural Networks (ANN)** 🧠
- Introduces deep learning principles. Builds, trains, and evaluates a multi-layer Perceptron (ANN) on the Breast Cancer dataset using `tensorflow`/`keras`.

### **Lab-10: Convolutional Neural Networks (CNN)** 👁️
- Explores computer vision techniques. Constructs a CNN model using Convolutional and MaxPooling layers to classify handwritten digits (MNIST dataset).

### **Lab-11: Natural Language Processing (NLP)** 💬
- A primer on text preprocessing. Covers essential concepts such as sentence and word tokenization, stop-words removal, morphological stemming, and lemmatization using the `NLTK` library.

---

## 🏁 Conclusion
This repository serves as a cumulative portfolio of hands-on Machine Learning competencies. By covering everything from traditional statistical regressions and standard classifiers to ensemble aggregation and deep/neural networks, it provides a well-rounded foundation necessary for addressing modern data challenges and building robust predictive pipelines.
